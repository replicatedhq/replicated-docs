# cluster port ls

List cluster ports for a cluster.

## Usage

```bash
replicated cluster port ls CLUSTER_ID [flags]
```

  <tr>
    <th width="30%">Flag</th>

    <td>`--output`</td>
    <td>string</td>
    <td>The output format to use. <strong>Value values:</strong> json, table or wide. <strong>Default:</strong> table</td>
