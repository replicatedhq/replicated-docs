# replicated policy ls

List RBAC policies

List all RBAC policies for your team.

```
replicated policy ls [flags]
```

### Aliases

```
ls, list
```

### Examples

```

  replicated policy ls

  # List policies in JSON format
  replicated policy ls --output json
```
