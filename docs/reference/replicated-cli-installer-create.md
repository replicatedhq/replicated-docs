# installer create

Create a new Kubernetes installer manifest by providing a YAML file for a cluster. For more information, see [Create An Installer](https://kurl.sh/docs/create-installer/) in the kURL documentation.

## Usage
```bash
replicated installer create [flags]
```

## Examples


```bash
cat longhorn.yaml
apiVersion: "cluster.kurl.sh/v1beta1"
kind: "Installer"
metadata:
  name: "longhorn-install"
spec:
  kubernetes:
    version: "1.21.x"
  weave:
    version: "2.8.x"
  prometheus:
    version: "latest"
  registry:
    version: "latest"
  containerd:
    version: "latest"
  kotsadm:
    version: "latest"
  ekco:
    version: "latest"
  minio:
    version: "latest"
  longhorn:
    version: "latest"
  metricsServer:
    version: "latest"
  goldpinger:
    version: "latest"

replicated installer create --yaml-file ./longhorn.yaml --promote Unstable
  • Creating Installer ✓
    • SEQUENCE: 2
  • Promoting ✓
    • Channel 1xy9tHhAwHyoc0HEybppxPpbPn6 successfully set to installer 2

```
