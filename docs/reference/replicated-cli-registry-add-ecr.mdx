# registry add ecr

Adds an ECR registry using an access key ID and secret access key.

## Usage

```bash
replicated registry add ecr [flags]
```
The following flags are supported:

<table>
  <tr>
    <th width="30%">Flag</th>
    <th width="20%">Type (if applicable)</th>
    <th width="50%">Description</th>
  </tr>
  <tr>
    <td><code>--accesskeyid</code></td>
    <td>string</td>
    <td>The access key ID to use when authenticating to the registry.</td>
  </tr>
  <tr>
    <td><code>--secretaccesskey</code></td>
    <td>string</td>
    <td>The secret access key to use when authenticating to the registry.</td>
  </tr>
  <tr>
    <td><code>--secretaccesskey-stdin</code></td>
    <td></td>
    <td>Takes the secret access key from stdin.</td>
  </tr>
  <SkipValidation/>
</table>
