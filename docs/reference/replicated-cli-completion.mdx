# completion 

Generates scripts for bash, fish, powershell, and zsh command completion.

## Usage

```bash
replicated completion [command] [flags]
```

The following `completion` commands are supported:

<table>
<tr>
  <th width="50%">Command</th>
  <th width="50%">Description</th>
</tr>
  <tr>
    <td><a href="replicated-cli-completion-bash"><code>replicated completion bash</code></a></td>
    <td>Generates <code>bash</code> completion script</td>
  </tr>
  <tr>
    <td><a href="replicated-cli-completion-fish"><code>replicated completion fish</code></a></td>
    <td>Generates <code>fish</code> completion script</td>
  </tr>
  <tr>
    <td><a href="replciated-cli-completion-powershell"><code>replicated completion powershell</code></a></td>
    <td>Generates <code>powershell</code> completion script</td>
  </tr>
  <tr>
    <td><a href="replciated-cli-completion-zsh"><code>replicated completion zsh</code></a></td>
    <td>Generates <code>zsh</code> completion script</td>
  </tr>
</table>
