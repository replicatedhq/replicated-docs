# replicated profile ls

List all authentication profiles

### Synopsis

List all authentication profiles configured in ~/.replicated/config.yaml.

The default profile is indicated with an asterisk (*).

```
replicated profile ls [flags]
```

### Examples

```
# List all profiles
replicated profile ls
```
