# default

Manage the default values used by other Replicated CLI commands.

## Usage

```bash
replicated default [command] [flags]
```
The following commands are available:

<table>
  <tr>
    <th>Command</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>clear</td>
    <td>Clears default value for the specified key.</td>
  </tr>
  <tr>
    <td>clear-all</td>
    <td>Removes all default values that are used by other commands.</td>
  </tr>
  <tr>
    <td>set</td>
    <td>Sets default value for the specified key.</td>
  </tr>
  <tr>
    <td>show</td>
    <td>Shows default values for the specified key.</td>
  </tr>
</table>
