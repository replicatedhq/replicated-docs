**Requirements:**

* Proxy installations require Embedded Cluster 1.5.1 or later with Kubernetes 1.29 or later.

* When installing behind a proxy, both HTTP and HTTPS proxy must be configured. You can set them using the `--http-proxy` and `--https-proxy` flags, or using the `http_proxy` and `https_proxy` environment variables. If neither flag is set, Embedded Cluster uses the corresponding environment variable. If you don't use an HTTP proxy, set the HTTP proxy to the same host and port as your HTTPS proxy server.
