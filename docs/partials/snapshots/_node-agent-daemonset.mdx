Configure the node-agent DaemonSet specification if your cluster uses one of the following Kubernetes distributions:
  * RancherOS 
  * OpenShift 
  * Microsoft Azure 
  * VMware Tanzu Kubernetes Grid Integrated Edition

For information about how to configure the node-agent DaemonSet for these distributions, see [Configure node-agent DaemonSet spec](https://velero.io/docs/v1.17/file-system-backup/#configure-node-agent-daemonset-spec) in the Velero documentation.
