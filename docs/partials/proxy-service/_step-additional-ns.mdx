If you are deploying Pods to namespaces other than the application namespace, add the namespace to the `additionalNamespaces` attribute of the Replicated Application custom resource. This ensures that the installer can provision the `imagePullSecret` in the namespace to allow the Pod to pull the image. For more information, see [`additionalNamespaces`](/reference/custom-resource-application#additionalnamespaces) in the Application custom resource reference.
