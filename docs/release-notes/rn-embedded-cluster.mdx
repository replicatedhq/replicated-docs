# Embedded Cluster Release Notes

## 0.0.0

Released on July 31, 2024

Support for Kubernetes: 0.0.0+k8s-1.29, 0.0.0+k8s-1.28

Support for KOTS: 0.0.0 and later

### New Features {#new-features-0-0-0}

* New upgrade wizard.

### Improvements {#improvements-0-0-0}

* Added better error messages to CLI output.

### Bug Fixes {#bug-fixes-0-0-0}

* Fixes an issue in Embedded Cluster upgrades.

### Known Issues {#known-issues-0-0-0}

* There is a known issue that causes the Admin Console to lose connection.
