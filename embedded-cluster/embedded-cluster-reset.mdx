# reset

This topic describes the options available with the Embedded Cluster `reset` command.

## Usage

```bash
sudo ./<app-slug> reset [flags]
```

## Flags

<table>
  <tr>
    <th width="30%">Flag</th>
    <th width="20%">Type</th>
    <th width="50%">Description</th>
  </tr>
  <tr>
    <td>`-f`, `--force`</td>
    <td></td>
    <td>Ignore errors encountered when resetting the node.</td>
  </tr>
  <tr>
    <td>`-h`, `--help`</td>
